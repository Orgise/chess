Help

This simple pixel editor is for quickly sketching out ideas.

The Pencil Button toggles between pencil mode and group selection mode (useful for large areas of colour). Unfortunately erasing, shading & highlighting are limited to pencil mode (for now).

Shading & Highlighting can be done by clicking the appropriate toggle next to the pencil tool.

New Canvas generates a new canvas based on window size, with two pixel size options.

Clear clears the current canvas.

Planned features: save, load, export, undo, shading, fill & basic objects.
